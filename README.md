# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/SamuelFR7/conselho-da-juventude.git
```

```bash
# Install dependencies
$ npm install

# Define env variables
$ cp .env.example .env

# Run the app
$ npm run dev
```

Go to http://localhost:3000/ to see the result.
